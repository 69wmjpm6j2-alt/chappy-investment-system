# CIS Step4D：買い場基準マスター 価格スケール再監査

## 結論
ユーザー指摘どおり、ニチレイ（2871.T）の基準はおかしかったため修正した。

## ニチレイ修正
- 旧案：3300 / 3000 / 2700
- 修正後：1650 / 1500 / 1350
- ステータス：locked_v1_2
- 理由：旧価格感が残った可能性が高く、買い場アラートが誤判定するリスクがあったため。

## 追加チェック結果
全銘柄を価格スケール/分割/逆分割/特殊条件の観点で再監査した。

### 高・中優先で注意する銘柄
- NOW｜ServiceNow｜SPLIT_SCALE_OK｜維持。ただしヘルスチェック対象｜2025年に5-for-1株式分割が発表されたため、90/80/70の価格帯は分割後スケールとして大きな矛盾なし。ただし日々の価格桁チェック対象。
- RGTI｜Rigetti｜DAILY_SCALE_CHECK_REQUIRED｜価格桁・逆分割・ティッカー変更チェックを毎回実施｜量子高ボラ。短期急騰急落により買い場判定が過敏になりやすい。
- MSTR｜MicroStrategy｜SPLIT_SCALE_OK｜維持。ただしヘルスチェック対象｜過去の株式分割後スケールとして大きな矛盾なし。ただしBTC条件を併用。
- POET｜POET Technologies｜DAILY_SCALE_CHECK_REQUIRED｜価格桁・逆分割・ティッカー変更チェックを毎回実施｜小型高ボラ。価格桁と流動性チェック必須。
- KITT｜Nauticus Robotics｜DAILY_SCALE_CHECK_REQUIRED｜価格桁・逆分割・ティッカー変更チェックを毎回実施｜超小型で逆分割・ティッカー/価格スケール変更リスクが高い。
- AXTI｜AXT｜DAILY_SCALE_CHECK_REQUIRED｜価格桁・逆分割・ティッカー変更チェックを毎回実施｜小型・低位株寄り。価格異常値チェック必須。
- OPTX｜Syntec Optics｜DAILY_SCALE_CHECK_REQUIRED｜価格桁・逆分割・ティッカー変更チェックを毎回実施｜超小型で逆分割・流動性・価格スケール変更リスクが高い。
- SPCX｜SpaceX/SPCX｜DAILY_SCALE_CHECK_REQUIRED｜価格桁・逆分割・ティッカー変更チェックを毎回実施｜取引可能ティッカー未確認。価格を作らない。
- 2871.T｜ニチレイ｜FIXED｜1650/1500/1350に修正済み｜v1.2で価格スケール修正済み。
- 4042.T｜東ソー｜VALUATION_REVIEW_NOT_SCALE_ERROR｜月次見直し対象｜東ソーは基準がやや高めに見える可能性あり。ただし価格スケール異常ではなく、バリュエーション見直し対象。
- 7821.T｜前田工繊｜VALUATION_REVIEW_NOT_SCALE_ERROR｜月次見直し対象｜前田工繊は既存基準を維持するが、成長鈍化時は本命価格を下げる候補。
- 1377.T｜サカタのタネ｜VALUATION_REVIEW_NOT_SCALE_ERROR｜月次見直し対象｜サカタのタネは業績サイクルがあり、初期基準は月次で再評価。
- 6287.T｜サトーHD｜VALUATION_REVIEW_NOT_SCALE_ERROR｜月次見直し対象｜サトーHDは基準がやや浅い可能性あり。ただし価格スケール異常ではなく、月次で業績/配当/ROE確認。

## 新しいガードレール
- 現在値が本命価格の0.25倍未満または4倍超の場合、買い場判定を停止する
- 株式分割・逆分割・ティッカー変更候補は data_health_log に記録する
- 超小型株は『強く買いたい』判定でも投資サイズを制限する
- MSTRは株価だけでなくBTC条件を併用する
- 買い場基準は毎日自動更新しない
